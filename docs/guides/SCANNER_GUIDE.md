### **Step 1: Adaptive Threshold** üéØ

**‡∏õ‡∏±‡∏ç‡∏´‡∏≤:** ‡∏´‡∏∏‡πâ‡∏ô‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏ï‡∏±‡∏ß‡∏°‡∏µ‡∏Ñ‡∏ß‡∏≤‡∏°‡∏ú‡∏±‡∏ô‡∏ú‡∏ß‡∏ô‡∏ï‡πà‡∏≤‡∏á‡∏Å‡∏±‡∏ô
- PTT: ‡∏ß‡∏±‡∏ô‡∏õ‡∏Å‡∏ï‡∏¥‡∏Ç‡∏∂‡πâ‡∏ô‡∏•‡∏á 0.5%
- DELTA: ‡∏ß‡∏±‡∏ô‡∏õ‡∏Å‡∏ï‡∏¥‡∏Ç‡∏∂‡πâ‡∏ô‡∏•‡∏á 2%

‡∏ñ‡πâ‡∏≤‡πÉ‡∏ä‡πâ threshold 1% ‡∏ó‡∏±‡πâ‡∏á‡∏´‡∏°‡∏î ‚Üí DELTA ‡∏à‡∏∞‡πÇ‡∏î‡∏ô‡∏ô‡∏±‡∏ö‡∏ó‡∏∏‡∏Å‡∏ß‡∏±‡∏ô (‡πÄ‡∏¢‡∏≠‡∏∞‡πÄ‡∏Å‡∏¥‡∏ô!)

**‡∏ß‡∏¥‡∏ò‡∏µ‡πÅ‡∏Å‡πâ:**
```python
def calculate_dynamic_threshold(df, lookback=90, multiplier=1.5):
    # ‡∏î‡∏∂‡∏á 90 ‡∏ß‡∏±‡∏ô‡∏•‡πà‡∏≤‡∏™‡∏∏‡∏î
    recent = df.tail(90)
    
    # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Standard Deviation
    std = recent['pct_change'].std()
    
    # Threshold = SD √ó 1.5
    threshold = std * 1.5
    
    # ‡∏à‡∏≥‡∏Å‡∏±‡∏î‡πÉ‡∏´‡πâ‡∏™‡∏°‡πÄ‡∏´‡∏ï‡∏∏‡∏™‡∏°‡∏ú‡∏• (0.5% - 5%)
    threshold = max(0.5, min(threshold, 5.0))
    
    return threshold
```

- ‚úÖ ‡∏´‡∏∏‡πâ‡∏ô‡πÄ‡∏™‡∏ñ‡∏µ‡∏¢‡∏£ ‚Üí threshold ‡∏ï‡πà‡∏≥ (‡πÑ‡∏ß‡∏ï‡πà‡∏≠‡∏Å‡∏≤‡∏£‡πÄ‡∏õ‡∏•‡∏µ‡πà‡∏¢‡∏ô‡πÅ‡∏õ‡∏•‡∏á)
- ‚úÖ ‡∏´‡∏∏‡πâ‡∏ô‡∏ú‡∏±‡∏ô‡∏ú‡∏ß‡∏ô ‚Üí threshold ‡∏™‡∏π‡∏á (‡∏Å‡∏£‡∏≠‡∏á‡πÄ‡∏â‡∏û‡∏≤‡∏∞‡∏Å‡∏≤‡∏£‡πÄ‡∏Ñ‡∏•‡∏∑‡πà‡∏≠‡∏ô‡πÑ‡∏´‡∏ß‡∏à‡∏£‡∏¥‡∏á‡πÜ)

### **Step 2: Consecutive Streak Detection** üî•

**‡∏Ñ‡∏≥‡∏ô‡∏¥‡∏¢‡∏≤‡∏°:**
- **Up Streak** = ‡∏Ç‡∏∂‡πâ‡∏ô‡∏ï‡∏¥‡∏î‡∏ï‡πà‡∏≠‡∏Å‡∏±‡∏ô > threshold
- **Down Streak** = ‡∏•‡∏á‡∏ï‡∏¥‡∏î‡∏ï‡πà‡∏≠‡∏Å‡∏±‡∏ô < -threshold
- **Quiet** = ‡∏≠‡∏¢‡∏π‡πà‡πÉ‡∏ô range

**Logic:**
```python
def detect_streak(df, threshold):
    streak = 0
    current_direction = None
    
    # ‡πÄ‡∏£‡∏¥‡πà‡∏°‡∏à‡∏≤‡∏Å‡∏ß‡∏±‡∏ô‡∏•‡πà‡∏≤‡∏™‡∏∏‡∏î ‡πÑ‡∏•‡πà‡∏¢‡πâ‡∏≠‡∏ô‡∏Å‡∏•‡∏±‡∏ö
    for i in range(len(df) - 1, -1, -1):
        change = df.iloc[i]['pct_change']
        
        # ‡∏ï‡∏£‡∏ß‡∏à‡∏™‡∏≠‡∏ö‡∏ó‡∏¥‡∏®‡∏ó‡∏≤‡∏á
        if change > threshold:
            direction = 'UP'
        elif change < -threshold:
            direction = 'DOWN'
        else:
            break  # Quiet day - ‡∏´‡∏¢‡∏∏‡∏î‡∏ô‡∏±‡∏ö
        
        # ‡∏ñ‡πâ‡∏≤‡∏ó‡∏¥‡∏®‡∏ó‡∏≤‡∏á‡πÄ‡∏î‡∏µ‡∏¢‡∏ß‡∏Å‡∏±‡∏ô
        if current_direction == direction:
            streak += 1 (‡∏´‡∏£‡∏∑‡∏≠ -1)
        else:
            break
    
    return streak
```
