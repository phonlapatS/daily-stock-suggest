## สรุปเหตุผลตามประเภท

### 1. หุ้นที่ออกจากตลาด/ถูกเพิกถอน (Delisted) - 5 หุ้น

#### INTUCH (Intouch Holdings) - SET
- **สถานะ:** ถูกเพิกถอนจากตลาด
- **เหตุผล:** Intouch Holdings อาจถูกควบรวมหรือเปลี่ยนชื่อบริษัท
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว

#### ESSO (Esso Thailand) - SET  
- **สถานะ:** ถูกเพิกถอนจากตลาด
- **เหตุผล:** Esso (Thailand) อาจถูกควบรวมหรือเปลี่ยนชื่อบริษัท
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว

#### STARK (Stark Corporation) - SET
- **สถานะ:** ถูกเพิกถอนจากตลาด (น่าจะล้มละลาย)
- **เหตุผล:** Stark Corporation มีปัญหาทางการเงินและน่าจะถูกเพิกถอนจากตลาด SET เนื่องจากล้มละลายหรือมีปัญหาทางการเงินร้ายแรง
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว

#### STEC (Sino-Thai Engineering) - SET
- **สถานะ:** ถูกเพิกถอนจากตลาด
- **เหตุผล:** Sino-Thai Engineering อาจถูกเพิกถอนหรือเปลี่ยนชื่อ/รหัสหุ้น
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว

#### SGEN (Seagen) - NASDAQ
- **สถานะ:** ถูกควบรวมและเพิกถอนจากตลาด
- **เหตุผล:** Seagen (SGEN) ถูก Pfizer ซื้อกิจการในเดือนธันวาคม 2023 และถูกเพิกถอนจาก NASDAQ หลังจากนั้น
- **วันที่เพิกถอน:** ธันวาคม 2023
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว

### 2. หุ้นที่ใช้ Exchange ผิด - 1 หุ้น

#### AZN (AstraZeneca) - NASDAQ
- **สถานะ:** ใช้ Exchange ผิด
- **เหตุผล:** AstraZeneca (AZN) ไม่ได้ซื้อขายบน NASDAQ แต่ซื้อขายบน London Stock Exchange (LSE) ด้วยรหัส AZN.L
- **การแก้ไข:** ควรใช้ AZN.L บน LSE แทน หรือลบออกหากไม่ต้องการติดตามหุ้น LSE
- **การดำเนินการ:** ลบออกจากรายชื่อ NASDAQ แล้ว

---

### 3. หุ้นที่มีปัญหาจาก Format - 1 หุ้น

#### 0981 (SMIC - Semiconductor Manufacturing International Corporation) - HKEX
- **สถานะ:** ปัญหา Format ของรหัสหุ้น
- **เหตุผล:** รหัสหุ้น HKEX อาจต้องใช้รูปแบบที่แตกต่างกัน บางระบบอาจไม่รองรับเลข 0 นำหน้า
- **การแก้ไข:** ลองใช้ "981" แทน "0981" หรือตรวจสอบรูปแบบที่ถูกต้องจาก HKEX
- **หมายเหตุ:** หุ้นนี้อาจยังมีอยู่จริง แต่ต้องใช้รหัสที่ถูกต้อง
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว (เนื่องจากไม่สามารถดึงข้อมูลได้)

### 4. หุ้นที่อาจยังมีอยู่แต่มีปัญหาเทคนิค - 1 หุ้น

#### WBA (Walgreens Boots Alliance) - NASDAQ
- **สถานะ:** อาจยังมีอยู่จริง
- **เหตุผล:** Walgreens Boots Alliance (WBA) ควรยังซื้อขายอยู่บน NASDAQ ปัญหาอาจเกิดจาก:
  - ปัญหาเทคนิคชั่วคราว

  - Rate limiting จาก TradingView
- **คำแนะนำ:** ควรตรวจสอบอีกครั้งในภายหลัง หรือลองใช้รูปแบบอื่น
- **การดำเนินการ:** ลบออกจากรายชื่อแล้ว (เนื่องจากไม่สามารถดึงข้อมูลได้ในขณะนี้)

## หมายเหตุ

- หุ้นที่ถูกเพิกถอนจากตลาดจะไม่มีข้อมูลใหม่ในระบบ TradingView หรือแหล่งข้อมูลอื่นๆ
- การลบหุ้นที่ไม่มีข้อมูลออกจะช่วยหลีกเลี่ยง "Survivorship Bias" ในการวิเคราะห์ข้อมูล
- หากต้องการติดตามหุ้นเหล่านี้ในอนาคต ควรตรวจสอบสถานะจากแหล่งข้อมูลทางการของตลาดหลักทรัพย์
